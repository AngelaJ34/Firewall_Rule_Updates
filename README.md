<h1>Microsoft Windows Defender Firewall IP Update List</h1>


<h2>Description</h2>
The project consists of a simple Python script that is designed to automatically block newly seen IP addresses block/blacklist malicious Ip Addresses in Windows Defender. This Python script will automatically update and block bad ip addresses therefore eliminating the need to perform this task manually in Windows Defender.
<br />

<h2>Objective</h2>
Block/blacklist Malicious Ip Addresses in Windows Defender using a Python script that automatically blocks newly seen IP addresses.

<h2>Languages and Utilities Used</h2>

- <b>Python</b> 
- <b>Windows Defender Firewall</b>

<h2>Environments Used </h2>

- <b>Windows 10</b> (21H2)
